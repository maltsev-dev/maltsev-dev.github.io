Text-based User Interface app. Inventory System

📚 Inventory System

Features

  1. Create and save new item
    • product type
    • quantity
    • price per unit
  2. Show all items

    • sales tax
  3. Delete item by ID
  4. DB located at const FILE_PATH: &str = "products.json";
